# resources-tool

### Показ лучших заводов для апа

- поставить свои текущие урони фабрик в `resources-tool.py`
- при необходимости поменять стоимости в `res.py`
- при необходимости добавить в фабрики в `resources-tool.py` модификатор скорости (например 1.15 для гастарбайтеров)
  - пример `FACTORY_CONCRETE = Factory('concrete', FACTORY_CONCRETE_CONTRACT, 258, 1.15)`
- запустить `resources-tool.py`
- ввести топ сколько показать
